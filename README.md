# 📧 AI Email Summarization Workflow (n8n)

This project is an automated workflow built with n8n that:

- Fetches incoming emails
- Summarizes email content using AI
- Assigns a priority score
- Outputs structured data

## 🔧 Technologies
- n8n
- OpenAI API (or whichever you used)
- Email integration (IMAP/Gmail)

## 🚀 How it works
1. Trigger: New email received
2. Process: AI summarizes the email
3. Scoring: Assigns importance score
4. Output: Structured response

## 📁 Files
- `workflow.json`: Exported n8n workflow

## 💡 Use Case
Helps prioritize emails and reduce information overload.
